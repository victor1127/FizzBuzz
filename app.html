[Fizz Buzz] A Coding Project

When it counts you Fizz Buzz. A favascript coding exercise using loops. Write a program that prints the numbers from 1 to 100 and for multiples of "3" display "Fizz" instead of the number and for multiples of '5' display "Buzz" . If it is multiple of both display "FizzBuzz".

Instructions

Enter a starting and ending values. Press the button and the values will appear.
